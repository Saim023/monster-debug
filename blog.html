Questions And Answers:

Question-1: Wtat is the difference between localStorage and sessionStorage??

Answer: sessionStorage is quite similar to localStorage. The difference is that while data in localStorage doesn't expire, on the other hand, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

Question-2: What is the difference between global scope and block scope??

Answer: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function. On the other hand, block scope is a variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

Question-3: How Event Loop works in javascript ??

Answer: Event Loop used for monitoring the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.

Question-4: How many ways to get undefined??

Answer: There are Eight ways to get undefined in javascript. (1) a variables that is not initialized will get undefined. (2) function with no return will get undefined. (3) a paramiter that is not passed will get undefined. (4) if a return has nothing on the right side will get undefined. (5) property that doesn't exists in an object will get undefined. (6) if you try to access an array elements outside of the index range will get undefined. (7) deleting an array inside of an array, will get undefined. (8) set a value directly undefined will get undefined.
